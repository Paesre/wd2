## Javascript

### Tipos de datos primitivos
- String
    ```js
    "string";
    'string';
    `string`;
    String('string');
    ```

- Number
    ```js
    1;
    1.01;
    0;
    -1.21;
    -12;
    Infinity;
    NaN;
    Number('10');
    ```
- Boolean
    ```js
    true;
    false;
    Boolean(0);
    ```

- Undefined
    ```js
    undefined;
    ```

- Null
    ```js
    null;
    ```
- Symbol
    ```js
    Symbol()
    ```

### Expresiones

**Matemáticas**
```js
4 + 3;
2 - 1;
1 * 7;
1 / 5;
4 % 2;
8 ** 2;
```

**Comparación**
```js
9 > 3;
1 < 5;
10 === 10;
'8' == 8;
10 !== 10;
4 != '4';
```
